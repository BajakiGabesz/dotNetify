## DotNetify Vue Demo - Stock Ticker

### Prerequisites

- (NodeJS)[https://nodejs.org/en/download/]
- .(NET 3.1 SDK)[https://dotnet.microsoft.com/download/dotnet/3.1]

### How to Run

```
npm i
dotnet run
```

Open `localhost:5000`.
